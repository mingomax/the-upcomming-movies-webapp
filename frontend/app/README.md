# UP Movies Web APP

Up Movies Web App is responsible to provides a easy website to list and view Upcoming films.


## How this was structured

This project was developed with [Angular](https://www.angular.io) and [Material Design for Angular Projects](https://material.angular.io).

### Directories

- **node_modules/**: local where all dependencies was install by package manager
- **src/enviroments/**: application configuration lives here
- **nginx/**: acessory to publish with Nginx Sewrver
- **src/**: source of project, included modules and components
- **.gitignore**: project git ignore files list
- **package-lock.json**: version description of depenecies and packages descriptions
- **package.json**: package and project descritption
